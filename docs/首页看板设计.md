## 数据来源（全是已有的，不多写一笔）

| 看板内容 | 来自哪里 | 
|---------|---------|
| 今日上班人员+工时 | 排班数据 |
| 本月收入 | 记账-各平台汇总 |
| 本月支出 | 记账-各项目汇总 |
| 本月利润 | 收入-支出，自动算 |
| 各平台排行 | 记账-按平台汇总 |
| 每日趋势 | 记账-按日期汇总 |
| 上月对比 | 上月收入/支出/利润 |
| 工资总额 | 工资模块 |
| 天气 | 免费API自动获取 |

## 老板每天用这个做什么

**早上打开看：**
- 今天谁上班？够不够人？
- 今天天气怎么样？影响出餐吗？

**晚上打开看：**
- 今天收入多少？比昨天好还是差？
- 这个月到现在赚了多少？

**月底看：**
- 这个月利润多少？
- 跟上月比是涨是跌？
- 哪个平台表现最好？
